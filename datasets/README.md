# Datasets

## Layout

- `shared/processed/`
  Shared Parquet exports produced by the Python janitor and consumed by the Rust athlete app.
- `calibration_20_shot/raw/side/`
  Side-view calibration videos.
- `calibration_20_shot/raw/angle45/`
  Forty-five-degree calibration videos.
- `calibration_20_shot/annotations/`
  Athlete metadata and shot-level tags.
- `calibration_20_shot/derived/`
  Generated features, metadata, and Parquet output.
